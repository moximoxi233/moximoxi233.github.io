CSS四个隐藏元素的方法

发表于 2019-09-25 | 分类于 CSS

当需要对元素进行隐藏时，我们常利用opacity、visibility、display、position、overflow、transform、 z-index来达到目的，它们的区别是？

CSS四个常见隐藏元素的方法

  1. opacity:0 将元素隐藏（设置透明度为0）

    • 依然占据文档流
    • 子元素无法显示
    • 可以点击

  2. visibility: hidden 将元素设置成隐藏

    • 依然占据文档流
    • 子元素可以显示 （前提设置：visibility: visible）
    • 可以点击

  3. display: none 将元素设置为不可见

    • 不占据文档流
    • 子元素不显示
    • 不可以点击

    🐷 相当于不存在

  4. position: absolute; left:-999px; top: -999px;

    • 不占据文档流
    • 子元素不显示
    • 不可以点击
    position: relative; left:-999px; top: -999px;
    • 不占据文档流
    • 子元素不显示
    • 不可以点击

其他隐藏元素的方法：

overflow:hidden 将移除元素隐藏

  • （隐藏溢出部分）不占据空间
  • 无法点击

position: absolute; z-index: -99; 设置元素z-index值

  • 不占据空间
  • 无法点击

transform: scale(0,0);

  • 依然占据文档流
  • 子元素无法显示
  • 不可以点击
