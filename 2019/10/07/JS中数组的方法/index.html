JS中数组的方法

发表于 2019-10-07 | 分类于 JS

javascript是一门充满活力、简单易用的一门脚本语言,它也是一门解释性、弱类型、基于对象、动态类型的语言。

注意 Js不是一门面向对象的语言

js中的数组

JavaScript中的数组比起其他语言中的数组有很大不同

​ js中的数组长度是可以动态改变的，同一个数组中也可以皴法不同数据类型的值。

数组属性

  • length 设置或返回数组元素的个数

  • prototype 向数组原型对象中添加属性和方法

  • constructor 返回对 Array 对象属性创建的函数。

    ​

数组对象中的方法

数组元素增减的常用方法

  1. pop() 删除并返回数组的最后一个元素

  2. shift() 删除并返回数组的第一个元素。

  3. push() 向数组的末尾添加一个或更多元素，并返回新的长度。

  4. unshift() 向数组的开头添加一个或更多元素，并返回新的长度。

  5. splice()

    • splice(a,b) 从下标为a的位置删除b个元素

      • 返回被删除的元素，且改变了原数组

    • splice(a, 0, t1, t2, t3…) 从下标为a的位置添加，t1,t2…

      • 改变原数组

    • splice(a, b, t1, t2…) 从下标未a的位置开始删除 b 项元素，并将 t1，t2插入到下标为a的位置

      • 返回被替代的那个元素
 - 改变原数组
数组中常用的方法

  1. toString() 把数组转换为字符串，并返回结果。

    • 返回值是将数组中的元素之间用逗号分隔
    • 不改变原数组

  2. slice() 选取数组的的一部分

    • 返回一个新数组。
    • 不改变原数组

    注意 slice(start,end)

    • start

      可选。规定从何处开始选取。如果是负数，那么它规定从数组尾部开始算起的位置。也就是说，-1 指最后一个元素，-2 指倒数第二个元素，以此类推。

    • end

      可选。规定从何处结束选取。该参数是数组片断结束处的数组下标。如果没有指定该参数，那么切分的数组包含从 start 到数组结束的所有元素。如果这个参数是负数，那么它规定的是从数组尾部开始算起的元素。

  3. concat() 连接多个数组，并返回连接后的数组

    注意 此方法不会改变现有的数组，而仅仅会返回连接数组

  4. findIndex() 返回符合传入测试（函数）条件的数组元素索引

    • findIndex() 返回符合条件的元素的索引位置，之后不会再调用执行函数。
    • 如果没有符合条件的元素返回 -1

    注意 findIndex() 对于空数组，函数是不会执行的, 且并没有改变数组的原始值

    ​ ==IE不支持==

  5. includes() 判断一个数组是否包含一个指定的值。

    • 如果是返回 true，否则false。

  6. join() 把数组的所有元素放入一个字符串

    • 通过指定的分隔符进行分隔的

  7. sort() 用于对数组的元素进行排序

    • 排序顺序可以是字母或数字，并按升序或降序。
    • 默认排序顺序为按字母升序
    • 如果返回值大于 0，则位置互换。如果返回值小于 0，则位置不变。

    注意 这种方法会改变原始数组！

  8. reverse() 颠倒数组中元素的顺序

    • 会改变原数组

  9. reduce() 将数组元素计算为一个值（从左到右）

    注意 reduce() 对于空数组是不会执行回调函数

  10. indexOf/lastIndexOf() 从 头/尾 搜索数组中的元素，并返回它最先出现的位置。

  1. isArray() 判断对象是否为数组

    • 如果对象是数组返回 true
    • 否则返回 false。

    注意 isArray()是内置对象Array的方法，要由它来调用

数组遍历

  1. every() 用于检测数组所有元素是否都符合指定条件（通过函数提供)

    • 数组中有一个元素不满足条件，则整个表达式返回 false。否则，返回true

    注意：every() 不会改变原始数组，不会对空数组进行正常检测

  2. some() 检测数组中的元素是否满足指定条件（函数提供）

    • 如果有一个元素满足条件，则表达式返回true ,剩余的元素不会再执行检测。否则，返回true

    注意：every() 不会改变原始数组，不会对空数组进行正常检测

  3. forEach() 用于调用数组的每个元素，并将元素传递给回调函数

    ​ forEach() 对于空数组是不会执行回调函数的

    注意：forEach() 不返回任何值，该方法只是将数组元素传递给回调函数

  4. map() 调用数组的每个元素，经过回调函数处理得到一个新的数组并返回

    注意：map() 不会对空数组进行检测，map() 不会改变原始数组

  5. find() 检测数组中的元素，返回通过测试（函数内判断）的数组的第一个元素的值(ES6)

    • 如果没有符合条件的元素返回 undefined

    注意： find() 对于空数组，函数是不会执行的。 并没有改变数组的原始值。

  6. filter() 检查指定数组中符合条件的所有元素,返回符合条件的元素组成的数组

总结

不会改变原数组的方法有
  • toString()
  • slice() 提取数组中的某个部分
  • concat()
  • findIndex()
  • includes()
  • join()
  • reduce()
  • indexOf/lastIndexOf()
  • isArray()
  • every()
  • some()
  • map()
  • find()
  • filter()
