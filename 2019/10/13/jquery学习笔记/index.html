Jquery学习笔记

发表于 2019-10-13 | 分类于 jQuery

jquery 是 javascript 库中的一种

javascript库：就是把一些浏览器兼容性代码或者常用的函数封装在一个 js 库，common.js可以看成js库

Jquery中的顶级对象

  • 浏览器中的顶级对象：window，浏览器和页面中所有东西都是window的
  • 页面的顶级对象：document，页面中某些东西是document
  • jQuery中的顶级对象：jQuery——-可以用$符号来代替。

如果想要使用jQuery中的属性和方法，那么需要jQuery.属性或者jQuery.方法()来使用（简单写法：$.属性,$.方法），大多数情况下jQuery中几乎都是方法，属性很少。

Jquery中几乎把dom中的事件都封装成了一个方法

  • dom中的注册事件

document.getElementById(“id属性值”).onclick()=匿名函数；

  • jquery中注册事件

$(“#id属性值”).click(匿名函数)

Jquery对象和DOM对象互转

  • DOM对象—转成jQuery对象，才能使用jQuery中的属性和方法。
  • DOM对象转Jquery对象===》$(DOM对象)
  • Jquery对象转DOM对象===》jQuery对象[0]

注：当两个对象想用另一个对象的属性和方法时，对象进行转换即可调用其属性和方法

jquery中的一些方法

  • .val() 首先是针对表单标签

    • 不传参数，表示的是获取value属性值
    • 传参数，设置元素的value属性值

  • .text() 成对的标签啊可以使用

    • 不传参数，表示的是获取标签中间文本内容
    • 传参数，设置标签中间的内容
  • .html() 成对的标签可以设置或者获取中间的html内容（跟.text()类似）

  • .css() 设置元素样式 .css("属性名字"，"值")

  • .show() 设置元素显示，括号中可以写数字，单位是毫秒

  • .hide() 设置元素隐藏，括号中可以写数字，单位是毫秒

  • .stop() 停止当前的动画

自定义动画方法
animate

$(selector).animate({params},[speed],[easing],[callback])

  • {params} 要执行动画的css属性

  • callback 动画执行完后立即执行回调函数【可选】

事件绑定区别

  1. 对象.事件名字（事件处理函数）;——>普通的写法
  2. 对象.bingd(“事件名称”,事件处理函数)

前两种方式只能为存在的元素绑定事件，后添加的元素没有事件

  1. 对象.delegate(“选择器”,”事件名字”,事件处理函数)
  2. 对象.on(”事件名字”,“选择器”,事件处理函数)

后两种方式都是父级元素调用方法，代理子级元素绑定事件

*注：对象.on(”事件名字”,事件处理函数)为自己绑定事件

元素事件解绑

unbind
  • 对象.unbind(“事件名称”)//会把这个对象对应的所有事件都给解绑
    • 对象.unbind(“事件名称1 事件名称2”)//会把这个对象对应的多个所有事件都给解绑
    • 对象.unbind()//没传入任何参数时，会把这个对象所有事件都给解绑
undelegate
  • 对象.undelegate(“选择器”,”事件名字”)//移除这个对象下的子级元素的所有对应事件
    • 对象.undelegate(”事件名字”)//移除这个对象下的子级元素的所有对应事件
    • 对象.undelegate(“选择器”,”事件名字1 事件名字2”)//移除这个对象下的子级元素的所有对应的多个事件
    • 对象.undelegate()//移除这个对象下的子级元素的所有事件
off
  • 对象.off(”事件名字”,”选择器”);//这个对象它对应子级元素的所有对应事件全部被解绑
    • 对象.off(”事件名字”);//这个对象以及它的子级元素的所有对应事件全部被解绑
    • 对象.off(”事件名字”,”**”);//这个对象以及它的所有子级元素的所有对应事件全部被解绑
    • 对象.off(”事件名字1 事件名字2 ”);//这个对象以及它的子级元素的所有对应的多个事件全部被解绑
    • 对象.off();//这个对象以及它的子级元素的所有事件全部被解绑
