JS中字符串的方法

发表于 2019-10-09 | 分类于 JS

JavaScript 字符串（String） 对象

String 对象用于处理已有的字符块。

String 对象属性

字符串属性

  • length 设置或返回数组元素的个数

  • prototype 向数组原型对象中添加属性和方法

  • constructor 返回对 String 对象属性创建的函数。

    ​

String对象常用方法

查找字符串方法

  1. charAt() 返回在指定位置的字符

  2. concat() 连接两个或更多字符串，并返回新的字符串。

    • 不会改变原字符串（str），但会返回一个新的字符串

  3. indexOf() 返回某个指定的字符串值在字符串中首次出现的位置。

    • 找到即返回该字符串的位置（第一个字符的下标）
    • 未找到，返回-1

  4. lastIndexOf() 从后向前搜索字符串，并从起始位置（0）开始计算返回字符串最后出现的位置。

  5. includes() 查找字符串中是否包含指定的子字符串。

    • 找到即返回true，否则返回false

  6. match() 查找找到一个或多个正则表达式的匹配。

    • 将匹配好的字符串保存到一个数组并返回这个数组

  7. repeat() 复制字符串指定次数，并将它们连接在一起返回。【ES6】

    • 返回复制后的字符串，但不会改变原字符串

  8. replace() 在字符串中查找匹配的子串， 并替换与正则表达式匹配的子串。

    • 不会改变原来的字符串，但会返回一个新的字符串
    • 第一个参数可以传入指定字符（该字符必须存在在原字符串当中，不然会报错），或者一个正则

  9. search() 查找与正则表达式相匹配的值。

    • 返回的是指定字符(正则匹配)首次出现的位置
    • 找不到指定字符则返回-1

  10. startsWith() 查看字符串是否以指定的子字符串开头。

字符串分割/提取方法

  1. slice() 提取字符串的片断，并在新的字符串中返回被提取的部分。

    • slice(start(包含), end(不包含))
    • 如果参数为负，则从字符串尾部开始

  2. split() 把字符串分割为字符串数组。

    • split() 方法不改变原始字符串。
    • 返回的是一个数组

  3. substtr() 从起始索引号提取字符串中指定数目的字符。

    • 参数1：提取字符的开始坐标，参数2：提取字符的个数

  4. substring() 提取字符串中两个指定的索引号之间的字符。

    • substring(start(包含), end(不包含))
    • 参数为非负整数
字符串基本操作

  1. toLocaleLowerCase() 根据本地主机的语言环境把字符串转换为小写。

    • 不改变原字符串，会返回一个新字符串

  2. toLocaleUpperCase() 根据本地主机的语言环境把字符串转换为大写。

    • 不会改变原来字符串，会返回一个新的字符串

  3. trim() 去除字符串两边的空白
